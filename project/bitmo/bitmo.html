BITMO

Bitmo was an idea to create an exchange for bitcoin, while this idea never came to fruition it stands as a good example of how to think of user flows and sketches and covert those into mockups. I think one of the best things about this project is that it was an *actual* idea that we came up with and tried to realize.

The following six sketches think about the way the screens will be layed out and how the user will click through various parts of the site.

We then proceeded to create a landing page mockup in Photoshop.

We also created a more fleshed out dashboard UI and login screen.

An icon was made.

And lastly, a wallpaper was created that incorporated geometric shapes to create a more "futuristic" effect.
